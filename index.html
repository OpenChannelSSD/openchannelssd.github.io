Open-Channel SSDs

Opening up SSDs for everyone

Welcome

A Open-Channel Solid State Drive is an SSD that exposes the physical flash storage through the logical address space, while keeping a subset of the internal features of common SSDs. The Open-Channel SSD project is a set of kernel extensions and user-space libraries that allows kernel and applications to directly interact with SSDs.
