Bringing Open-Channel SSDs to a data center near you!

Strict I/O Isolation

Provide end-to-end IO isolation by physical partitioning.

Predictable Latency

No more guessing for when an IO completes. Perform data placement optimized for your workloads.

Software Defined NVM

Manage your non-volatile memory as a block device, through a file-system or inside your application. Eliminate device DRAM in the progress.

Documentation

Open-Channel SSD is a full-stack initiative from system specification, operating system support, user space management tools, I/O libraries, to examples of their use.

Getting Started

  • Open-Channel Solid State Drive Specification
    Download 1.2 final
    Access current work-in-progress edition
    The LightNVM Open-Channel specification is an open standard, aiming to incorporate the needs of designers, implementers, and users of Open-Channel SSDs.

  • Linux Kernel Support
    https://github.com/OpenChannelSSD/linux
    Implementation of operating system support for Open-ChannelSSDs in the Linux kernel. It is the core of identifying, managing, and using Open-Channel SSDs.

  • nvme-cli
    https://github.com/linux-nvme/nvme-cli
    Open-Channel SSD management available through standardilized cli tools.

  • liblightnvm
    https://github.com/OpenChannelSSD/liblightnvm
    Integrate Open-Channel SSDs into your applications using liblightnvm. It simplifies developments and enable one to use predefined data striping techniques.

  • RocksDB Open-Channel SSD Support
    https://github.com/OpenChannelSSD/rocksdb
    Implements support in RocksDB for directly accessing Open-Channel SSDs.

Visit GitHub to see all projects

Hardware

Open-ChannelSSDs are an emerging technology which is available from various vendors

  • CNEX Labs Westlake SDK.
    LightNVM SDK may be ordered for academic purposes. Please write to matias@cnexlabs.com with your research description and inquire for availability.

  • EMC Dragon FireBoard Open-Channel OX Controller

  • Radian Memory Systems RMS-325

  • Lite-On AD2 Open-Channel SSD
    Lite-On Storage announced, at Flash Memory Summit 2016, support for Open-channel in their AD2 series

  • IIT Madras SSD Controller
    https://bitbucket.org/casl/ssd-controller
    An open-source implementation of a NVMe controller in BlueSpec. Can run on Xilinx FPGA's such as Artix 7, Kintex 7, and Vertex 7.

Please send a pull request to add other supported platforms.
